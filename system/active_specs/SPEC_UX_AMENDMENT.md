# SPEC Amendment — UX Enhancements Corrections

## 1. Death Save Stabilization (Feature #3)

The spec says stabilized means "conscious but at 0 until healed." This is wrong.

**Correct rule:** Stabilized = unconscious, no longer making death saves. The character remains unconscious at 0 HP. They regain consciousness only when healed to 1+ HP. If not healed, they naturally regain 1 HP after 1d4 hours.

Update the Death Save section: Three successes → "Stabilized — unconscious but no longer dying. You need healing to wake up (or you regain 1 HP in 1d4 hours)."

## 2. Absorb Elements in Quick Rules (Feature #9)

Remove "Absorb Elements" from the common reactions list in the Quick Rules Reference. It is from Xanathar's Guide, not the PHB. Replace with "Hellish Rebuke" (Warlock reaction spell, PHB) or simply remove the entry and keep the list to: Opportunity Attack, Shield, Uncanny Dodge, Counterspell.
